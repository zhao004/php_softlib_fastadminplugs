# Softlib

## 介绍

Softlib 是一个基于 [FastAdmin](https://doc.fastadmin.net/) 的开源软件库管理系统插件。

![Softlib](./assets/images/20260122-103053.png)

### 教程

- 视频教程：<https://www.fastadmin.net/video.html>

## 下载当前发布版本

- 进入 Releases（最新版本）：[releases/latest](../../releases/latest)
- 在 Assets 中下载 `softlib-vx.y.z.zip`（以实际版本号为准）

## 自动发布

当推送符合 `v1.0.0` 格式的标签（如 `v1.2.3`）时，GitHub Actions 会自动将源码打包为 `softlib-<tag>.zip` 并创建 GitHub
Release。

工作流文件：`.github/workflows/release.yml`
